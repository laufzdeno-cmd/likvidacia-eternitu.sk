# AGENTS.md

- Projekt je slovenský web pre likvidáciu eternitu/azbestu.
- Všetky verejné texty majú byť po slovensky.
- Nepoužívaj falošné firemné údaje.
- Ak chýbajú údaje, použi TODO placeholder a urob ho editovateľný v adminovi.
- Priorita je bezpečnosť, rýchlosť, SEO, UX a konverzia dopytov.
- Dotazník musí zostať krátky.
- Cenová ponuka sa nikdy nesmie odoslať bez schválenia adminom.
- Uploadované súbory zákazníkov nesmú byť verejné bez schválenia.
- Fotky z terénu musia ísť cez schválenie pred publikáciou.
- SEO dáta musia zodpovedať viditeľnému obsahu.
- Každú zmenu validuj testami/buildom, ak je to možné.
- Chráň osobné údaje.
- Pri neistote zvoľ bezpečnejšiu a jednoduchšiu implementáciu.
