# likvidacia-eternitu.sk

Statický web pre doménu `likvidacia-eternitu.sk` postavený na Vite + React + TypeScript.

## Lokálne spustenie

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

Produkčný výstup je v priečinku `dist`.

## GitHub Pages

Deploy rieši workflow `.github/workflows/deploy.yml`. V nastaveniach repozitára nastav GitHub Pages source na `GitHub Actions`. Custom doména je `likvidacia-eternitu.sk`.
